# Netflix Movie Recommendation System

This project is a high-quality, student-led machine learning project to build a personalized movie recommendation system using the Netflix Prize Dataset.

## Project Structure
- `data/`: Contains raw and processed subsets of the dataset.
- `notebooks/`: Exploratory Data Analysis and initial model prototyping.
- `src/`: Modular Python scripts for data processing, models, and evaluation.
- `app/`: Streamlit dashboard for interactive exploration of recommendations.

## Setup Instructions
```bash
pip install -r requirements.txt
```
